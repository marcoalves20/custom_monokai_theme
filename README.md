# JetBrains IDE Color Themes

A collection of custom color schemes for JetBrains IDEs (IntelliJ IDEA, PyCharm, WebStorm, CLion, GoLand, etc.).

## Available Themes

| Theme | File | Description |
|-------|------|-------------|
| Sublime Monokai Custom | `Sublime_Monokai_Custom.jar` | Sublime Text-inspired Monokai color palette |
| Catppuccin | `Catppuccin.Theme-3.5.2-base.jar` | Soothing pastel theme for the high-spirited |

## Installation

1. Download the `.jar` file for your preferred theme
2. Open your JetBrains IDE
3. Go to **File > Settings > Plugins** (or **Preferences > Plugins** on macOS)
4. Click the gear icon and select **Install Plugin from Disk...**
5. Select the downloaded `.jar` file
6. Restart the IDE
7. Go to **File > Settings > Editor > Color Scheme** and select your theme

## Theme Details

### Sublime Monokai Custom

- Sublime Text-inspired Monokai color palette
- Optimized for readability and reduced eye strain
- Dark background with vibrant syntax highlighting

### Catppuccin

- Pastel color palette with warm, cozy tones
- Four flavor variants available (Latte, Frappe, Macchiato, Mocha)
- Community-driven theme with active development
