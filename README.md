# Custom Monokai Theme

A custom Monokai color scheme for JetBrains IDEs (IntelliJ IDEA, PyCharm, WebStorm, etc.).

## Installation

1. Download `Sublime_Monokai_Custom.jar`
2. Open your JetBrains IDE
3. Go to **File > Settings > Plugins**
4. Click the gear icon and select **Install Plugin from Disk...**
5. Select the downloaded `.jar` file
6. Restart the IDE
7. Go to **File > Settings > Editor > Color Scheme** and select **Sublime Monokai Custom**

## Features

- Sublime Text-inspired Monokai color palette
- Optimized for readability and reduced eye strain
- Works across all JetBrains IDEs
